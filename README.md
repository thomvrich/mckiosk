This is my project to improve the UI and UX of the Kiosk for the OU Math Center. Some features I added include changing the button on the landing page from "Get Started" to "Check In or Check Out" to improve the UX. In terms of UI improvements on the second screen, I added the color of each math to the color room it is in in the Math Center. Additionally, I added text for colorblind users. Another feature I added was an admin screen that lets front desk workers toggle the hours-of-operation display, a custom message, and optional snow for a little holiday cheer. The prototype is moving up the chain after I met with internal stakeholders who liked it, and some parts of it will make it to production.
